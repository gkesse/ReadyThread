# ReadyThread

**ReadyThread** est une bibliothèque de fonctions C/C++ pour le développement d'applications Multithreading.  
C'est une bibliothèque développée en C/C++ et maintenu par 
[Gerard KESSE](http://5.48.215.193:8855/presentation/ "Accédez à mon site web (ReadyDev)").

Les concepts arbodés sur les Bases :  
* [Processus](https://github.com/gkesse/ReadyThread/tree/master/Processus "Processus")  

# Technologies

Le développement de la bibliothèque **ReadyThread** a nécessité les technologies suivantes :

* Langage : **C/C++**
* IDE : **Qt Creator**
* Editeur de Text : **Notepad++**
* Outil de Construction : **QMake, NMake**
* Patron de Conception : **Singleton**
* Cible : **Windows, UNIX**
